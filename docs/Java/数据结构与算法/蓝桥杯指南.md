## 1、基本赛制

- 蓝桥杯是一个`OI`赛制的算法竞赛。

> OI的基本规则是：
>
> + 提交每道题之后都不会有任何的反馈；
> + 每道题都会有多个测试点，根据每道题通过的测试点数量获得相应的分数;
> + 不限制每道题的提交次数，仅以最后一次提交为准
> + 比赛过程中看不到任何实时排名，最后按照总得分进行排名
